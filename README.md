# Tic Tac Toe Game

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features
- Two player game (X vs O)
- Winner detection
- Reset / New Game functionality

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Play
- Click on any box to place your move
- First player starts with X
- The game announces the winner when a pattern is matched
- Click "New Game" to reset the board
